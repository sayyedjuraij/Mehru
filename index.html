Core Modules

Five Powerhouses

A

Unique QR Code System

Shop-specific QR codes securely mapped to Shop IDs in MSSQL. Mobile-optimized — no app required. Instant, encrypted access to dynamic feedback forms.

  • Unique generation per shop
  • Secure DB mapping
  • Mobile-first landing
B

AI Reviewer Flow

5-step pipeline: dynamic form → Python NLP → structured review generation → clipboard copy → Google Review redirect. Zero friction for the customer.

  • Dynamic questionnaire per shop
  • AI-generated natural review text
  • One-tap Google Review submit
AI
C

Shop Owner Dashboard

Full analytics suite: satisfaction scores, sentiment breakdown, trend charts, AI-driven insights. Owners manage their own questionnaires in real time.

  • Sentiment: Positive / Neutral / Negative
  • Trend charts over time
  • AI common feedback insights
D

Super Admin Panel

Global control: shop accounts, QR mappings, questionnaire controls, subscription monitoring, and usage analytics across all registered shops.

  • RBAC: Super Admin vs Shop Owner
  • Subscription engine
  • Cross-shop usage analytics
🛡
E

Payments & Security

Full Razorpay integration for subscription purchases. RBAC authentication, end-to-end API encryption, and industry-standard data storage security.

  • Razorpay webhook handling
  • JWT + RBAC auth
  • AES-256 encryption
